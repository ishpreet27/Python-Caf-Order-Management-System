# Python Café – Order Management System

A simple **Tkinter-based Café Ordering System** that allows users to place food orders through a GUI and automatically stores all order details in an Excel file using **OpenPyXL**.

---

## Features

* GUI-based café menu using Tkinter
* Select multiple items and quantities
* Automatic bill calculation
* Generates Order ID automatically
* Saves orders in an Excel file
* Timestamp stored for each order
* Option to add more items to the same order

---

## Menu Items

* Pizza – Rs 40
* Pasta – Rs 50
* Burger – Rs 60
* Salad – Rs 70
* Coffee – Rs 80

---

## Technologies Used

* Python
* Tkinter (GUI)
* OpenPyXL (Excel file handling)
* Datetime
* OS module

---

## Installation

Install required library:

```bash
pip install openpyxl
```

---

## How to Run

```bash
python main.py
```

---

## Excel Storage

Orders are stored automatically in:

```
orders.xlsx
```

Each record contains:

* Order ID
* Item
* Quantity
* Price
* Total Bill
* Timestamp
